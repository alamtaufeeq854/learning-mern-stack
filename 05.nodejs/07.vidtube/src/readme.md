# VideoTube Backend

A backend project built with Node.js, Express.js, and MongoDB following a scalable REST API architecture.

## Tech Stack

* Node.js
* Express.js
* MongoDB
* Mongoose
* JWT
* Bcrypt
* Multer
* Cloudinary

## Features

* User Authentication
* Video Management
* Comments & Likes
* Channel Subscriptions
* Playlist Management
* Secure JWT Authentication
