# 🛣️ React Router

A React application built to understand client-side routing using React Router.

## 📚 Concepts Covered

- React Router
- BrowserRouter
- Routes & Route
- Nested Routes
- Dynamic Routing
- Navigation with Link & NavLink
- Layout Components

## 🛠️ Tech Stack

- React
- Vite
- React Router DOM
- JavaScript

## ▶️ Run Locally

```bash
npm install
npm run dev
```

## 📌 Purpose

Built for learning client-side routing and navigation in React applications.
