# ⚡ Basic Vite

A basic React project created using Vite to understand the Vite development environment and project setup.

## 📚 Concepts Covered

- Vite Setup
- React Project Structure
- Hot Module Replacement (HMR)
- Development Server
- Build Process

## 🛠️ Tech Stack

- React
- Vite
- JavaScript

## ▶️ Run Locally

```bash
npm install
npm run dev
```

## 📌 Purpose

Built for learning the React + Vite development workflow.
