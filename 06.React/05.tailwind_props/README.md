# 🎨 Tailwind Props

A React project built to understand component reusability using props and styling with Tailwind CSS.

## 📚 Concepts Covered

- React Props
- Reusable Components
- Component Composition
- Tailwind CSS

## 🛠️ Tech Stack

- React
- Vite
- Tailwind CSS
- JavaScript

## ▶️ Run Locally

```bash
npm install
npm run dev
```

## 📌 Purpose

Built for learning reusable React components and Tailwind CSS styling.
