# 💱 Currency Converter

A React application that converts currencies using real-time exchange rates fetched from an API.

## 📚 Concepts Covered

- Custom Hooks
- useState Hook
- useEffect Hook
- API Integration
- Component Reusability
- State Management

## 🛠️ Tech Stack

- React
- Vite
- JavaScript
- Tailwind CSS
- Exchange Rate API

## ▶️ Run Locally

```bash
npm install
npm run dev
```

## 📌 Purpose

Built for learning custom hooks, API integration, and state management in React.
