# 🌙 Theme Switcher

A React application that allows users to switch between light and dark themes using the Context API.

## 📚 Concepts Covered

- Context API
- createContext
- Context Provider
- useContext Hook
- Theme Management
- Conditional Styling

## 🛠️ Tech Stack

- React
- Vite
- Tailwind CSS
- JavaScript

## ▶️ Run Locally

```bash
npm install
npm run dev
```

## 📌 Purpose

Built for learning global theme management using React Context API.
