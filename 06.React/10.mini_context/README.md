# 🌐 Mini Context

A React application built to understand the Context API for managing and sharing state across components without prop drilling.

## 📚 Concepts Covered

- Context API
- createContext
- Context Provider
- useContext Hook
- Global State Management
- Component Reusability

## 🛠️ Tech Stack

- React
- Vite
- JavaScript

## ▶️ Run Locally

```bash
npm install
npm run dev
```

## 📌 Purpose

Built for learning React's Context API and managing shared state across components.
