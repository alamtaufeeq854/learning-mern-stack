# 🔢 Counter App

A simple React counter application built to understand state management using the `useState` Hook.

## 📚 Concepts Covered

- useState Hook
- State Management
- Event Handling
- Component Re-rendering

## 🛠️ Tech Stack

- React
- Vite
- JavaScript

## ▶️ Run Locally

```bash
npm install
npm run dev
```

## 📌 Purpose

Built for learning React state management with the `useState` Hook.
