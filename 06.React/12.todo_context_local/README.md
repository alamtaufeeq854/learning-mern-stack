# ✅ Todo App

A React Todo application with persistent storage using the Context API and Local Storage.

## 📚 Concepts Covered

- Context API
- useState Hook
- useEffect Hook
- CRUD Operations
- Local Storage
- State Management

## 🛠️ Tech Stack

- React
- Vite
- JavaScript
- Tailwind CSS

## ▶️ Run Locally

```bash
npm install
npm run dev
```

## 📌 Purpose

Built for learning global state management and persistent data storage in React.
