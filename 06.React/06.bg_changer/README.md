# 🎨 Background Color Changer

A simple React application that changes the background color dynamically using buttons and the `useState` Hook.

## 📚 Concepts Covered

- useState Hook
- Event Handling
- Dynamic UI Updates
- Conditional Styling

## 🛠️ Tech Stack

- React
- Vite
- Tailwind CSS
- JavaScript

## ▶️ Run Locally

```bash
npm install
npm run dev
```

## 📌 Purpose

Built for learning state-driven UI updates and dynamic styling in React.
