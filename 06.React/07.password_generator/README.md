# 🔐 Password Generator

A React application that generates secure random passwords with customizable length, numbers, and special characters.

## 📚 Concepts Covered

- useState Hook
- useEffect Hook
- useCallback Hook
- useRef Hook
- Event Handling
- Clipboard API

## 🛠️ Tech Stack

- React
- Vite
- Tailwind CSS
- JavaScript

## ▶️ Run Locally

```bash
npm install
npm run dev
```

## 📌 Purpose

Built for learning React Hooks and creating a dynamic password generator.
